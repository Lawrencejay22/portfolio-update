# Simple Portfolio - Lawrence Jay

A clean, modern, and responsive portfolio website showcasing Lawrence Jay's skills and experience.

## Features

- **Responsive Design**: Works perfectly on all devices
- **Smooth Animations**: AOS (Animate On Scroll) library integration
- **Interactive Elements**: Hover effects, smooth scrolling, typing animation
- **Modern UI**: Clean design with professional color scheme
- **Contact Form**: Direct email integration
- **Skills Visualization**: Animated progress bars
- **Timeline**: Experience and education timeline

## Sections

1. **Home**: Hero section with typing animation
2. **Services**: Web Development, UI/UX Design, Graphic Design
3. **Skills**: Technical and professional skills with progress bars
4. **Education**: Academic background and certifications
5. **Experience**: Professional timeline
6. **Contact**: Contact form with email integration

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- JavaScript (ES6+)
- AOS Library
- Font Awesome Icons

## Quick Start

1. Open `index.html` in your browser
2. Or serve locally with a simple HTTP server

## Customization

- Update personal information in `index.html`
- Modify colors and styling in `style.css`
- Add more functionality in `script.js`
- Replace `LJAY-pic.jpg` with your own photo
