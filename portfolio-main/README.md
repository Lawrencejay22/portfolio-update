## Quick Start

### Main Website (Recommended)
Navigate to the `website/html/` folder and open `index.html` in your browser.

### Simple Portfolio
Navigate to the `portfolio/` folder and open `index.html` in your browser.

## Features

- **Responsive Design**: Works on all devices
- **Multi-page Navigation**: Home, Skills, Contact, Resume
- **Social Media Integration**: Links to Facebook, Instagram, YouTube, GitHub
- **Professional Services**: UI/UX Design, Graphic Design, Web Development
- **Contact Integration**: Direct email contact functionality
